Lecture 11 : Switch Case Control Statements
...

  • We use switch statements as it is less messy as compared to if else statements.
  • Switch case is faster than if else.
  • Switch case works only with integer and character, float will not work here.
#include <stdio.h>

int main()
{
    int a=2;    //we could've taken the value of `a` through scanf as well.
    switch (a)
    {
        case 2:
            printf("Value is 2");

            break; // breaks the operation.

        case 3:
            printf("Value is 2");
        default:
            printf("Nothing matched");  //default statement is printed when no case is satisfied.
    }
}

Rules for Switch Statement :

  1. Switch expression must be an int or charac.
  2. Case value must be an integer or a character.
  3. Case must come inside switch.
  4. break statement is not a must.

Some examples of valid and invalid statements

  1. Valid :
int x;
switch(x)
{
	 //cases+code
}
  1. Invalid :
float x;   // invalid because `x` can only be an integer or a character.
switch(x)
{
	 //cases+code
}

Lecture 18 : Typecasting
...

Typecasting
...

  • Typecasting is used to convert one data type to another data type.
  • Syntax : (type) value;
  • e.g. :
#include <stdio.h>
int main()
{
    int a=3;
    float b= 54/5;
    printf("value is %f", b);
    return 0;
}

Here output will be : 10.000000 but we know that 54/5 = 10.800000.

  • This is because 54 and 5 are integers and if we perform an arithmetic operation b/w two integers the output will also be an integer.
    • To solve this issue we'll use typecasting : e.g. :
#include <stdio.h>
int main()
{
    int a=3;
    float b= (float)54/5;  // typecasting syntax : (type) value;
    printf("value is %f", b);
    return 0;
}

Now the output will be 10.800000

  • Another e.g. :
#include <stdio.h>
int main()
{
    int a=3;
    float b= 54.456234;
    printf("value is %d", (int) b);
    return 0;
}

We were supposed to use %f instead of %d as we are printing the value of b but since we've already typecasted value of b as int so we can use %d here.
