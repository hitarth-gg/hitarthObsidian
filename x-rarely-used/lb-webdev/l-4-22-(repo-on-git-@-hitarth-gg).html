align-self :
...

The align-self CSS property overrides a grid or flex item's align-items value. In Grid, it aligns the item inside the grid area. In Flexbox, it aligns the item on the cross axis.

  • align-items — controls alignment of all items on the cross axis. align-self — controls alignment of an individual flex item on the cross axis

  • alignItems and alignSelf have the same functionality per se. The only difference being, alignItems applies the alignment on to it's children while alignSelf applies the alignment to itself. This is useful when you want to change the alignment of one child. alignSelf can override the alignItems value for specific items.
